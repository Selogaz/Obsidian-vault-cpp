## Теория А6 — Простейшие уравнения

### Основные виды уравнений

| Вид | Форма | Решение |
|-----|-------|---------|
| **Линейное** | $ax + b = 0$ | $x = -\frac{b}{a}$ |
| **Квадратное** | $ax^2 + bx + c = 0$ | $x = \frac{-b \pm \sqrt{D}}{2a}$, $D = b^2 - 4ac$ |
| **Дробно-рациональное** | $\frac{P(x)}{Q(x)} = 0$ | $P(x) = 0$, $Q(x) \neq 0$ |
| **Иррациональное** | $\sqrt{f(x)} = g(x)$ | $f(x) = g^2(x)$, $g(x) \geq 0$ |
| **Показательное** | $a^{f(x)} = a^{g(x)}$ | $f(x) = g(x)$ при $a > 0, a \neq 1$ |
| **Логарифмическое** | $\log_a f(x) = \log_a g(x)$ | $f(x) = g(x)$, $f, g > 0$ |
| **Тригонометрическое** | $\sin x = a$, $\cos x = a$, $\tan x = a$ | см. единичный круг |

### ОДЗ (Область допустимых значений)

- **Знаменатель** $\neq 0$
- **Под корнем** $\geq 0$ (чётная степень)
- **Под логарифмом** $> 0$
- **Основание логарифма** $a > 0$, $a \neq 1$
- **Под тангенсом** $\cos x \neq 0$

### Алгоритм решения

1. Определить тип уравнения
2. Записать ОДЗ
3. Выбрать метод решения
4. Найти корни
5. Проверить корни по ОДЗ
6. Записать ответ

### Частые ошибки

- Потеря корня при делении на выражение с переменной
- Добавление лишних корней при возведении в степень
- Забыть про ОДЗ
- Не учитывать ограничения тригонометрических функций
