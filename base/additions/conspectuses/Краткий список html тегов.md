# 📋 Введение в html
<!--комментарий-->
парные/одиночные теги
`<br>` - перенос строки
`<img src="work.png">` - картинка
`<br />` - альтернативная запись одиночного тега(устаревшая)

## Атрибуты тегов

`<p class="text-red"></p>`
`<div id="logo"></div>`
`<p title="текст в подсказке(hover)"></p>` - hover без css

- между атрибутами 1 пробел, а также между тегом
- все значения атрибутов указываются в двойных кавычках

## Категории тегов по типу содержания

- Metadata content - метаданные для браузеров, находятся в теге `<head></head>
- Flowconent - потоковый контент, теги находятся в `<body></body>`
- Sectioning content - крупные смысловые разделы
- Heading content - заголовки
- Phrasing content - сам текст документа, мелкий текст мельче абзаца
- Embedded conent - встраиваемый контент (изображения, видео, аудио)
- Interactive content - то, с чем взаимодействует пользователь

`<!DOCTYPE html>` - декларация типа документа. Говорит какой тип html используется.
*стили* подключаются в теге `<head>`
```html
<head>
	<link rel="stylesheet" href="style.css">
</head>
```
тег `<meta>` - ceo оптимизация, внутри тега `<head>`
```html
<meta charset="utf8">
<meta name="description" content="Описание страницы">
```
тег `<script src="scripts.js"></script>` указывается в самому низу тега `<body>`

валидация зависит от того, какой DOCTYPE указан

## Блочные элементы

`<p>, <h1>, <div>, <ul>`

*Особенности блочных элементов*
- всегда переносятся на новую строку
- можно задать ширину, высоту, отступы
- по умолчанию ширина равна ширине окна

## Строчные элементы

`<a>, <span>, <strong>, <em>

Особенность `<span>` - применяется для выделения текста, который нужно менять в css.(внутри divа)

*Особенности строчных элементов*
- идут друг за другом в 1 строке
- отступы только слева и справа
- ширина и высота равна содержимому элемента, изменить с помощью стилей нельзя
Задача строчных элементов - оформление текста

## Смысловые разделы

`<article></article>` - комментарий, статья, виджет погоды. Желателен заголовок внутри
`<section></section>` - смысловой раздел. В отличие от артикл нельзя отделить от страницы
`<nav></nav>` - обозначение ссылок навигации
`<aside></aside>` - боковая панель. Содержимое косвенно связано с основным

## Теги для текста

### Тег абзаца
`<p></p>`
### Теги выделения полужирным
`<strong></strong>` - определяет важность текста
`<b></b>` - без придания важности
### Теги выделения курсивом
`<em></em>` - придание акцента
`<i></i>` - не делает акцент на важности

### Теги переноса строки и разделитель
`<hr>` - горизонтальная линия-разделитель. Можно задать через css
`<br>` - переносит текст на новую строку. Можно оформлять контакты. Плохая практика для разметки абзацев.

### Теги индекса
используются для написания единиц измерений и простых формул
`<sup></sup>` - текст в виде верхнего индекса
`<sub></sub>` в виде нижнего индекса

### Тег преформатированный текст
По умолчанию браузер удаляет пробелы и переносы строк, но этот тег позволяет этого избежать
`<pre></pre>` - текст без изменений со стороны браузера

### Тег вставки кода
`<code></code>` - для кода, лол

### Теги цитаты
`<blockquote></blockquote>` - цитата как отдельный блок
`<q></q>` - для коротких цитат в предложениях. Текст внутри тега автоматически обрамляется кавычками
`<cite></cite>` - выделить источник цитаты, название или автора.

### Тег аббревиатуры
`<abbr title="Расшифровка"></abbr>`. Подсказка появляется при наведении

### Тег контактной информации\
`<address></address>` внутри можно использовать другие теги. Курсив по умолчанию

### Тег выделения текста
`<mark></mark>` - выделяет текст визуально(;желтым цветом)

### Заголовки и подзаголовки
`<h1></h1>` - обязательно использовать хотя бы 1 на сайте. Важно для ceo.
6 видов заголовков. Ниже 3 не используются

### Неупорядоченный или маркированный список
```html
<ul>
	<li></li>
	<li></li>
</ul>

<ul type="disc">...</ul> <!--атрибут type задает вид маркера-->
<ul type="circle">...</ul>
<ul type="square">...</ul>
```

### Упорядоченный или нумерованный список
```html
<ol>
	<li></li>
	<li></li>
</ol>

<ol start="5">...</ol> <!--атрибут start задает с какого числа начинается нумерация-->
```

### Многоуровневый список
Уровень вложенности не ограничен. Можно использовать оба типа списков
```html
<ul>
	<li>Мото</li>
	<li>Авто
		<ul>
			<li>Легковые</li>
			<li>Грузовые</li>
		</ul>
	</li>
	<li>Водный</li>
</ul>
```

### Список определений
`<dl>...</dl>` - открывает и закрывает список
`<dt>...</dt>` - содержит название термина
`<dd>...</dd>` - определение термина
```html
<dl>
	<dt>Сингулярность</dt>
	<dd>Этот термин довольно многолик</dd>
	
	<dt>Нейронная сеть</dt>
	<dd>Совокупность нейронов и спинного мозга</dd>
</dl>
```
## Тег ссылка
```html
<a href="http://google.com">Найди меня</a>
```
Виды адресов:
- Абсолютные
- Относительные
- Указывающие на страницу
- На файл
- Изображение
- Содержащие якорь(наверх)

атрибут `title` добавляет подсказку при наведении

атрибут `target` - способ открытия ссылки. Используется со значением `_blank` если открывать нужно в новом окне.
другие значения атрибута `target`:
- `_self`
- `_parent`
- `_top`

`<img src="img/house.jpg">

другие атрибуты изображения:

- width - ширина (px или %)
- height - высота

## Теги таблиц
`<table>...</table>` - открывает и закрывает таблицу
`<tr>...</tr>` - добавляет строку
`<td>...</td>` - добавляет столбец

ВАЖНО! В каждой строке должно быть одинаковое количество ячеек

```html
<table>
	<tr>
		<td>Ячейка 1</td>
		<td>Ячейка 2</td>
	</tr>
</table>
```
для указания заголовка столбцу или строке используется `<th>`. Он располагается внутри `<tr>` и аналогичен `<td>` . Текст внутри `<th>` выравнивается *по центру* и показывается *жирным*
```html
<table>
	<tr>
		<th>Магазин</th>
		<th>Количество товара</th>
	</tr>
	<tr>
		<td>Пятерочка</td>
		<td>345</td>
	</tr>
</table>
```
`<caption></caption>` - заголовок таблицы. Позицию и выравнивание можно изменить стилями
```html
<table>
	<caption>Учет по магазинам</caption>
	<tr>
		<th>Магазин</th>
		<th>Количество товара</th>
	</tr>
	<tr>
		<td>Пятерочка</td>
		<td>345</td>
	</tr>
</table>
```
атрибут тега td`colspan` - объединяет столбцы
атрибут тега th`rowspan` - объединяет строки
```html
<table>
	<tr>
		<td colspan="2">Ячейка на 2 столбца</td>
	</tr>
	<tr>
		<td>Ячейка 1</td>
		<td>Ячейка 2</td>
	</tr>
</table>
```
атрибуты тега `<table>`:
- атрибут `border` - толщина границы
- атрибут `cellspacing` - внешние отступы ячеек
- атрибут `cellpadding` - внутренние отступы ячеек

## Теги формы
`action` - задает адрес куда отправлять данные формы
`method` - задает метод передачи данных(get, post). По умолчанию `get`
`autocomplete="off"` - убрать подсказки при вводе данных
```html
<form action="https://tproger.com/login/" method="post">
<!--поля формы-->
</form>
```
get - данные передаются через url адрес
post - передаются через сервер

### Тег input
- атрибут `type` задает тип поля и влияет на то, как поле отображается в браузере. По умолчанию тип `text`. Другие *значения атрибута* `type`:
	- `password` - значение поля будет отображаться звездочками
	- `checkbox` - создает поле галочку. Атрибут `checked` ставит галочку сам.
	- `radio` - переключатель. Тип поля для выбора варианта. Обычно это группа полей, которая связана именем. У поля переключатель *всегда должен быть атрибут* `value`. Чтобы сделать общую группу, нужно задать каждому *одинаковый атрибут* `name`
	- `file` - добавляет поле для загрузки файла. При вставке поля в форму требует обязательно наличие у формы атрибута `enctype` со значением `multipart/form-data`.
	- `hidden` - скрывает поле от пользователя. При этом данные с этого поля передаются на сервер.
	- `button` - кнопка . В обычных случаях не требует атрибута `name`. Можно заменить тегом кнопки.
	- `email`
	- `phone`
	- `date`
	- `time`
	- `submit` - *отправляет на сервер* и перезагружает страницу
	- `reset` - очищает введенные данные
```html
<label> Есть машина
	 <input type="radio" name="user-car" value="yes" checked>
 </label>
<label> Нет машины
	<input type="radio" name="user-car" value="yes">
</label>
```
- атрибут `name` необходим, чтобы считать информацию с поля на сервере
- `id` - *уникальный* id поля. Для связи поля и подписи, для удобного выбора поля в коде JavaScript
- `value` - значение поля по умолчанию
- `placeholder` - подсказка для ввода
- `disabled` - выключить чекбокс(сделать серым)

### Тег label
Подпись к полю. Атрибут `for` *связывает label и input* через атрибут `id`
```html
<form action="https://tproger.com/login/" method="post">
	<label> Имя <input type="text" name="user-name"></label>
</form>

<form action="https://tproger.com/login/" method="post">
	<label for="user-name-id">Имя</label>
	<input id="user-name-id" type="text" name="user-name">
</form>
```

### Раскрывающийся список или селект
```html
<select name="color">
	<option value="red">Красный</option>
	<option value="green">Зеленый</option>
</select>
```
На сервер отправляется значение атрибута `value`. Если такой не задан, то отправится текст в `<option>`

### Селект с множественным выбором
В отличие от радиокнопок в селектах можно выбирать несколько вариантов из списка. Для этого тегу нужно добавить атрибут `multiple`
```html
<select name="cars[]" multiple>
	<option value="lada">Lada</option>
	<option value="bmw" selected>BMW</option>
	<option value="lexus">Lexus</option>
</select>
```
### Многострочное поле ввода
Используется для передачи большого текста. Атрибуты:
- `name`
- `id`
- `rows` - высота поля в строках
- `cols` - ширина поля в символах
```html
<textarea name="comment" rows="100" cols="100">
	Очень много текста с добрыми отзывами!<!--текст по умолчанию-->
</textarea>
```

## HTML5 теги
html5 теги в некоторых браузерах *может не работать*. Лучше использовать js и css
```html
<input list="cars">
<datalist id="cars"><!--выпадающий список-->
	<option value="Audi">
	<option value="BMW">
	<option value="Volvo">
</datalist>

<details>
	<summary>Показать</summary><!--кнопка показывает скрытый текст-->
	<p>Скрытый текст</p>
</details>

<audio controls><!--кнопки для контроля музыки-->
	<source src="asdf.mp3" type="audio/ogg; codecs=vorbis">
	Тег audio не работает в вашем браузере!
</audio>
```

### Прогресс бар

```html
<progress value="20" max="100">20%</progress>

<meter value="20" max="100" low="10" high="90">Топливо</meter><!--альтернатива progress-->
```
`low` и `high`меняют цвет индикатора при достижении указанного значения
