**Статус zotero**
—
Характеризует на каком этапе рабочего процесса находится источник
- 📥 inbox - новый, необработанный материал. Автоматизируется с помощью `Actions & Tags`
- 🟥 todo - запланировано к обработке/изучению
- 🟦 wip - в процессе активной работы
- 🟩 done - работа с материалом завершена
- ✖ drop - ненужный/неподходящий
- ⭐ favorite - *дополнительный* тег. Ключевые источники, исследовать в первую очередь
- 💎 obsidian - *дополнительный* тег. Есть заметка в Obsidian. Автоматизируется с помощью плагина Zotero `MarkDB-Connect`
про способ применения данных тегов написано в [[жизненный цикл источников]]
