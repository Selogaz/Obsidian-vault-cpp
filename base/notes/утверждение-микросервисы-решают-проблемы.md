# Утверждение: Microservices решают ключевые проблемы ML deployment

**Problems + Microservice solutions:**

| Проблема | Микросервисное решение |
|----------|------------------------|
| GPU scaling | Separate inference service with GPU autoscaling |
| Model versioning | Model Registry microservice |
| Team independence | ML team → Python service, Backend → Java |
| Failure isolation | Circuit breaker prevents cascade |
| Polyglot architecture | TensorFlow Serving (C++) + Spring Boot (Java) |
| Independent deployment | Model updates without app redeploy |

**Real-world examples:**
- Netflix: ML recommendations via microservices
- Uber: Michelangelo platform — microservices for ML lifecycle
- Airbnb: Bighead — distributed ML infrastructure

**Вывод:** Microservices provide architectural pattern that addresses ML-specific challenges.
