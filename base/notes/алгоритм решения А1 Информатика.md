искать точки среди меньшинства:
- Если точек с 2 путями меньше - начинать с них. Если с 3 меньше - наоборот.
- выбрать любые 2 точки из этого меньшинства
- определить соседи ли эти точки - проверить пути друг в друга.
- проверять по порядку выходящие из тех, что выбрали самыми первыми
- искать общие точки для определения номера

Если точек с 2 и 3 выходами одинаковое количество - ищем уникальное сочетание:
- 2 точки с 3 выходами, у которых есть пути друг в друга или 2 точки с 3 выходами.
- ищем все соседние точки у одной из выбранных
- затем ищем соседние у второй выбранной, проверяя общие точки с первой для определения номера
