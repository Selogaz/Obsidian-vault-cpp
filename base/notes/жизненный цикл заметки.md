# Жизненный цикл заметки

Пять стадий зрелости заметки, от сырой идеи до устоявшегося знания.

```
🌱 Seed → ♨️ Incubator → 🌿 Fern → 🌲 Evergreen
```

## Стадии

| Тег | Иконка | Стадия |
|-----|--------|--------|
| `note/basic/seed` | 🌱 | Сырая идея-зародыш. Только что записана, не сформирована |
| `note/basic/incubator` | ♨️ | Активно разрабатывается: добавляются связи, структура |
| `note/basic/fern` | 🌿 | Почти готова, но ещё есть куда расти |
| `note/basic/evergreen` | 🌲 | Зрелая заметка. Самодостаточна, хорошо связана, отражает устоявшееся понимание |
| `note/basic/primary` | 📝 | Заметка без жизненного цикла — справочная, процессная, сопроводительная |

## Ранжирование

База заметок (`home/databases/notes.base`) сортирует их по зрелости:

1. 🌲 evergreen — наиболее развитые
2. ♨️ incubator
3. 🌿 fern
4. 🌱 seed
5. 📝 other (primary) — без стадии

## Как использовать

1. **seed** — записал сырую идею. Не сформирована, не структурирована, может быть даже не до конца осмыслена
2. **incubator** — начал разрабатывать: добавляешь связи с другими заметками, структурируешь, уточняешь формулировки
3. **fern** — идея оформилась, почти готова. Можно показывать другим, но ещё есть шероховатости
4. **evergreen** — полностью готова. Самодостаточна, не требует контекста, хорошо связана с другими заметками. Отражает твоё устоявшееся понимание темы
5. **primary** — для заметок, которым жизненный цикл не нужен: описание процесса, справочная информация, конспект без амбиций развития

## Принципы

- Не обязательно проходить все стадии. Идея может пойти сразу в `fern`, если родилась уже сформированной
- Стадия `evergreen` — не финал навсегда. Если понимание изменилось, можно откатить до `incubator`
- `primary` — честный выбор: «эта заметка не идея, а инструмент»
