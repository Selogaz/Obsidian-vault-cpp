**БЭМ**
—
Блок, Элемент, Модификатор - компонентный подход к веб-разработке. В его основе лежит принцип разделения интерфейса на независимые блоки. Он позволяет облегчить разработку и повторно использовать существующий код. Код становится понятнее другим разработчикам
- в css использовать *исключительно классы*, избегая id и тегов
- нужно четко понимать где *блоки*, а где *элементы блоков*
	- блок - та часть сайта, которая может существовать как отдельный компонент, независимо
		- каждый блок должен иметь свой класс, хорошо ему подходящий
	- модификатор - состояние объекта(активная ссылка, спрятанный блок)
