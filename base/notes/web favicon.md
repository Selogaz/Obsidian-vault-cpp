**favicon**
—
`favicon.ico` достаточно для большинства случаев. Но Next.js рекомендует все три, потому что разные платформы предпочитают разные форматы/размеры:

- **`favicon.ico`** — вкладки браузера, классические закладки. Все десктопные браузеры.
- **`apple-icon.png` (180×180)** — экран iPhone/iPad когда сайт добавлен «На главный экран» через Safari. **iOS игнорирует favicon.ico** для этой цели.
- **`icon.png` (512×512)** — Android Chrome «Добавить на главный экран», PWA-манифест, превью в некоторых поисковиках/мессенджерах.

**Если тебе достаточно вкладок и закладок в десктопных браузерах** (никто не добавляет сайт на главный экран телефона) — можно смело удалить оба PNG. Сайт работать будет идентично, просто на iOS-устройстве при «Add to Home Screen» Apple возьмёт скриншот вкладки вместо нормальной иконки.
