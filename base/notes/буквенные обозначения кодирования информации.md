Канонические буквы для задания 7 ЕГЭ информатики (кодирование текста, изображений, звука, передача). Совпадают с нотацией сцены [[7-кодирование информации]] — выдумывать ничего не нужно.

> [!tip] Главное правило
> Жёсткого «буквенного ГОСТа» нет: в пособиях символы слегка гуляют (объём `V` или `I`; частота `ν`/`v`/`f`). **В условии задачи ЕГЭ величины задаются прямо в тексте — буквы берёшь оттуда.** Для своих заметок держись таблицы ниже.

## Словарь обозначений

| Величина | Буква | Единица |
|---|---|---|
| Информационный объём | **V** (в части пособий **I**) | бит |
| Количество пикселей | **K** (= ширина × высота) | шт |
| Глубина цвета / глубина кодирования | **i** | бит (на пиксель / на отсчёт) |
| Количество цветов / мощность алфавита | **N** (= 2^i) | шт |
| Частота дискретизации | **ν** (ню) | Гц |
| Число каналов | **k** (моно 1, стерео 2, квадро 4) | — |
| Количество символов | **n** | шт |
| Скорость канала | **q** | бит/с |
| Время | **t** | с |

## Формулы

| Что | Формула |
|---|---|
| Текст | $V = n \cdot i$,  $N = 2^{i}$ |
| Изображение (растр) | $V = K \cdot i$,  $N = 2^{i}$ |
| Звук | $V = \nu \cdot i \cdot k \cdot t$ |
| Передача | $V = q \cdot t \Rightarrow t = V / q$ |

`V = K·i` и `N = 2^i` — один и тот же приём для текста и картинки (мощность ↔ вес), см. [[карта связей — ЕГЭ информатика]]. Ошибки по теме → [[журнал ошибок — ЕГЭ информатика]]. Метод работы: [[цикл умной подготовки к ЕГЭ]].

## Где «стандарт»

- **ФИПИ** (fipi.ru) — кодификатор, спецификация, демоверсия, открытый банк заданий: стандарт *содержания*.
- **Поляков** (kpolyakov.spb.ru), учебники Поляков–Еремин / Босова — де-факто стандарт *обозначений и формул*.[^1]

[^1]: Формулы и обозначения сверены: [ege-study.ru — задание 7](https://ege-study.ru/ru/ege/materialy/informatika/polnyi-kurs-podgotovki-k-ege-po-informatike/zadanie-7/).
