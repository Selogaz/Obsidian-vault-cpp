**Правила работы с источниками**
—
- Пожалуйста, объясни как правильно работать с информацией с помощью источников.Например, нужно ли сначала создать конспект по нескольким(всем известным?) источникам и только потом превращать их в отдельные заметки?Или стоит разбивать на заметки сразу, по мере прохождения материала?Стоит ли мне создавать отдельную заметку с карточками(как я тебя попросил недавно) или лучше карточки вплетать в материал по мере изучения?

Это методологический вопрос о работе с источниками, отвечу напрямую.

**Два основных подхода:**

1. **Конспект → Разбивка** — сначала один файл с общими заметками по всему материалу, потом отдельные заметки. Хорошо для сложных тем, где нужно увидеть целостную картину перед структурированием.
2. **Сразу на заметки** — по мере прохождения каждого источника создаёшь отдельные заметки. Быстрее, но риск получить разрозненные заметки без связей.

**Рекомендую гибрид:**
- Для каждого источника (книга, статья) — создай заметку-источник в `sources/`
- Ключевые идеи из него сразу разбивай на отдельные заметки в `base/notes/`
- Когда накопилось достаточно заметок из разных источников — создай обобщающую заметку (мета-заметку или иерархию)

**Про карточки:**
Отдельные заметки лучше. Создавай заметку-карточку в `base/flashcards/` и ставь ссылку `[[]]` на исходную заметку с идеей. Вплетать в текст заметки неудобно — карточка должна быть автономной для Anki.
