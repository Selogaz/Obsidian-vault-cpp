Теги Codeforces для подготовки к собеседованиям, сгруппированные по уровню:

**Level 1 — Базовые:**
- `implementation` — внимание, синтаксис
- `math` — GCD, LCM, модульная арифметика
- `strings` — работа со строками
- `sorting` — сортировки, компараторы

**Level 2 — Core Algorithms:**
- `binary search` — бинарный поиск
- `two pointers` — two pointers, sliding window
- `prefix sum` — префиксные суммы
- `hashing` — хеш-таблицы
- `stack` — стек, балансировка
- `queue` — очереди, BFS

**Level 3 — Структуры данных:**
- `trees` — DFS/BFS на деревьях
- `graphs` — графы, Dijkstra

**Level 4 — Продвинутые:**
- `dp` — динамическое программирование
- `bitmasks` — битовые операции
