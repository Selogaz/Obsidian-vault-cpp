**перенести егэ на день между задачами**
—
Вместо того чтобы делать ЕГЭ после работы (когда мозг уже не тянет), перенести его на дневной перерыв между рабочими задачами.

Работа удалённая, не привязана ко времени. Между задачами есть окна по 15–30 минут — сейчас они уходят на скроллинг/провисание. Заменить их на ЕГЭ-информатика (лёгкий предмет) как переключение между сложными задачами.

**Плюсы:** мозг ещё не устал, работа не отвлекает (свои окна), нет чувства «ещё учиться после работы».
