**kebab-case**
—
стиль именования переменных с помощью дефиса
```css
:root {
	--main-color: grey;
}
```
Используется в [[css]], т.к. соответствует общей стилистике языка
Существуют другие стили:
- camelCase - принят в [[java]]
- snake_case - вроде как в python
