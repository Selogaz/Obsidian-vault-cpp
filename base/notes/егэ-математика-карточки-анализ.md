Карточки с решениями — правильный подход, но нужно добавить анализ ошибок.

**Что работает:**
- Карточки в проекте содержат подробные решения (метрические отношения, теорема Пифагора)
- Space repetition работает для долговременной памяти
- Несколько способов решения одной задачи (A1 — 3 способа)

**Что нужно добавить:**
- Теги "трудная/лёгкая" на карточки
- После каждого пробника — фиксация ошибок с указанием:
  - Номер задачи
  - Тема
  - Причина ошибки
  - Балл потерян/получен
