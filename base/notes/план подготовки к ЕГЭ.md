## ⚠️ Критические ошибки, которых стоит избегать

1. **«Нагнать всё за выходные»** → выгорание. Лучше 1 час ежедневно + 3 часа в выходные.
2. **Игнорировать часть 1** → 62 балла дают первые 12 заданий по математике. Отработайте их до автоматизма.
3. **Не писать сочинения** → русский «провалится» из-за одного несданного сочинения. Пишите каждые 2 недели.
4. **Гнаться за №18–19 по математике** → для бюджета достаточно уверенно решать до №15. Сложные задачи — только если база крепкая.

## 💡 Ваше преимущество

Ваш технический бэкграунд (фронтенд, Linux, системное администрирование) даёт **огромное преимущество по информатике**:
- №24/25 (алгоритмы) — будут даваться легче благодаря опыту с логикой программирования
- №16 (базы данных) — проще освоить, если знакомы с данными
- №9 (электронные таблицы) — тривиально при работе с данными

**Фокус:** доведите информатику до 80+ баллов — это компенсирует возможные сложности с математикой.

## 🚨 Срочные действия (19–29 февраля)

1. **Сегодня:** Пройдите 3 диагностических варианта (математика, информатика, русский) на РешуЕГЭ.
2. **Завтра:** Проанализируйте ошибки → составьте список «болевых тем».
3. **До 25 февраля:** Отработайте 2 слабые темы по математике + 1 по информатике.
4. **26–28 февраля:** Пройдите ещё 1 полный вариант по каждому предмету → замерьте прогресс.
5. **29 февраля:** Составьте детальный план на март с учётом графика 2/2.

> ✅ **Главное:** Не пытайтесь выучить всё. Фокус на части 1 + 2–3 сложных заданиях части 2 даст 70+ баллов — этого достаточно для бюджета в КубГТУ/ДГТУ по заочной форме.
