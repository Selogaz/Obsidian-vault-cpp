Здесь важное уточнение: **LM Studio не умеет генерировать картинки**. Это программа исключительно для текстовых моделей (LLM).

Для генерации изображений на твоем железе (RTX 3060 12GB + Manjaro) нужно использовать движки на базе **Stable Diffusion** или **Flux**. Твоя видеокарта идеально подходит для этого (12 ГБ VRAM — это золотой стандарт для локальной генерации).

Вот что тебе нужно установить вместо LM Studio для картинок:

### 1. Программное обеспечение (Интерфейс)

Тебе нужен "веб-интерфейс" для работы с моделями диффузии. Для Linux и RTX 3060 рекомендую два варианта:

**Вариант А: Fooocus (Самый простой)**

- **Почему:** Работает по принципу "нажал кнопку — получил красиво". Минимум настроек, автоматически подбирает параметры. Очень похож на Midjourney, но локально.
- **Для кого:** Если хочешь быстро получать результат без изучения сложных меню.
- **Установка на Manjaro:**

		bash
		
		1
		
		2
		
		3
		
		4
		
		5
		
		6
		
		_(Убедись, что установлены драйверы NVIDIA и CUDA)_.

**Вариант Б: Stable Diffusion WebUI Forge (Продвинутый)**

- **Почему:** Это оптимизированная версия популярного Automatic1111. Работает быстрее и лучше управляет видеопамятью.
- **Для кого:** Если нужен полный контроль над процессом, лорами, контролнетами и т.д.
- **Установка:** Аналогично через git clone репозитория `lllyasviel/stable-diffusion-webui-forge`.

### 2. Модели для генерации (Checkpoints)

Поскольку ты ищешь модели "без цензуры", тебе нужно скачивать их с сайта **Civitai.com**. Там сообщество выкладывает тысячи дообученных моделей.

**Что искать для RTX 3060 12GB:** Тебе нужны модели архитектуры **SDXL** (Stable Diffusion XL). Они дают лучшее качество, чем старые SD1.5, и твоя карта их тянет.

1. **Juggernaut XL** — универсальная, фотореализм, высокая детализация.
2. **RealVisXL** — отличный реализм, люди выглядят как на фото.
3. **Pony Diffusion V6 XL** — **Важно:** Это самая популярная модель для контента "без ограничений" (NSFW, аниме, специфические запросы).
		- _Нюанс:_ Она требует особого стиля промптов (теги через скобки, оценка качества `score_9, score_8_up` и т.д.).
		- _Где брать:_ На Civitai по запросу "Pony V6".

### 3. Как настроить под 12 ГБ VRAM

В настройках интерфейса (Fooocus или WebUI):

1. **Resolution:** Для SDXL оптимально **1024x1024** или **896x1152**. Не ставь сразу 4K, карта может перегреться или выдать ошибку OOM (Out of Memory).
2. **Batch Size:** Ставь **1**. Если поставить больше, генерация нескольких картинок сразу может переполнить память.
3. **Command Line Args (для WebUI):** Добавь аргументы для оптимизации: `--xformers --opt-sdp-attention --medvram` (Хотя на 3060 12GB часто хватает просто `--xformers`).

### 4. Итоговая инструкция для Manjaro

1. **Установи зависимости:**

		bash
		
		1
		
		_(Убедись, что драйверы NVIDIA версии 535+ установлены)_.
2. **Скачай Fooocus** (как указано выше).
3. **Зайди на Civitai**, зарегистрируйся (чтобы видеть скрытый контент, если нужно), найди модель **Juggernaut XL** или **Pony Diffusion**.
4. **Скачай файл `.safetensors`** (весит 6–7 ГБ).
5. **Положи файл** в папку `Fooocus/models/checkpoints`.
6. **Запусти** `python entry_with_update.py`.
7. **Генерируй.**

**Резюме:** LM Studio оставь для кода и учебы. Для картинок ставь **Fooocus** + модель **Juggernaut XL** (для реализма) или **Pony V6** (для свободы действий). Твоя RTX 3060 12GB справится отлично.
