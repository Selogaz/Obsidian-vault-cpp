**cвойства CSS анимации**
—
- `animation-name` - определяет имя анимации ([[css keyframes|keyframes]]), в котором описывают кадры анимации
- `animation-duration` - время одного цикла анимации
- `animation-timing-function` - ускорение анимации
- `animation-delay` - задержка между временем загрузки элемента и временем начала анимации
- `animation-iteration-count` - количество повторений анимации
- `animation-direction` - возможность при каждом повторе анимации идти по альтернативному пути, либо сбросить все значения и повторить анимацию
