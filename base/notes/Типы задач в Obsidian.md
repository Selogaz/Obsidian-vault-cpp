**Типы задач в Obsidian**
—
```zsh
## Типы задач в Obsidian GTD

| Тег | Когда использовать |
|-----|-------------------|
| `#task/inbox` | Быстрый захват, необработанная задача |
| `#task/next_action` | Приоритетная задача для немедленного выполнения |
| `#task/one-off` | Простая задача, одно действие |
| `#task/multistep` | Проект с подзадачами |
| `#task/waiting_for` | Делегировано или ждёшь внешнего действия |
| `#task/regular` | Повторяющаяся задача |
| `#task/idea` | Мимолётная мысль или вдохновение |
| `#task/reference` | Ресурс для изучения/чтения |
| `#task/someday` | Когда-нибудь, без конкретного срока |

```
