**Информатика А2 найти соответствие между столбцами и переменными с помощью таблицы истинности**
—

Логическая функция _F_ задаётся выражением 
x∧¬w∧(¬z∨y)
На рисунке приведён фрагмент таблицы истинности функции _F_, содержащий **все** наборы аргументов, при которых функция _F_ истинна.
Определите, какому столбцу таблицы истинности функции _F_ соответствует каждая из переменных _w_, _x_, _y_, _z_.
![[Подготовка к ЕГЭ 2026-02-24.png]]

**Решение:**
1. Посмотри на функцию. Если там много `AND` (∧∧), найди переменные, которые должны быть строго 0 или строго 1.
2. Найди в таблице столбцы, состоящие только из 0 или только из 1.
3. Оставшиеся переменные подставь в оставшуюся часть формулы и проверь по одной-двум строкам таблицы.
