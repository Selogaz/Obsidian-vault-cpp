**viewport height**
—
- [[web dvh]] — динамическая, меняется когда браузер прячет/показывает UI → причина проблемы
- `svh` — статическая малая (как будто браузерный UI всегда виден) → стабильно, но не использует появляющееся пустое место сверху, когда UI скрывается
- `lvh` — статическая большая (как будто браузерный UI всегда скрыт) → стабильно, но обрезается под адресной строкой
