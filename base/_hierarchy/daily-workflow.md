# Дневной рабочий процесс: Якоря дня

## Расписание по времени

```
Утро (на улице, до ~10:00):
  → English — 30 мин

День (между рабочими задачами):
  → ЕГЭ — 30 мин (чередование Математика / Информатика)

Вечер (после работы):
  → Vault upkeep — 15 мин

Основная работа (Подземелья Максвелла) — безлимит, приоритет
```

## Принципы

1. **Якорь ≠ задача.** Не рекерринг, не дедлайн. Чекбокс в daily note — отметил и пошёл дальше.
2. **30 мин — минимальная доза.** Меньше — не считается для ЕГЭ (15 мин — пустая трата). Больше — приветствуется.
3. **Пропуск ≠ срыв.** Пропустил день → вернулся на следующий без чувства вины.
4. **Правило «не два подряд».** English нельзя пропускать два дня подряд.
5. **Сначала работа.** Если работа плотная — ЕГЭ не впихивать насильно.
6. **Воскресенье — vault-day.** Никаких ЕГЭ, только Obsidian + Zotero.
