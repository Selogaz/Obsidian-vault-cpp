## Альтернативы для opencode

### 1. Другие модели в opencode
Opencode поддерживает 75+ провайдеров моделей. Можно переключиться на:
- **Claude Sonnet** — бесплатный tier от Anthropic
- **GPT-4o/GPT-5** — через OpenAI
- **Gemini** — через Google
- **Локальные модели** — через Ollama (Qwen 2.5 Coder, CodeQwen)

### 2. Альтернативные CLI-агенты
- **Claude Code** ($20-200/мес) — терминальный агент от Anthropic, лучший для сложных задач
- **OpenAI Codex** — CLI с открытым исходным кодом, бесплатный в некоторых режимах
- **Aider** — open-source, работает с любыми моделями

### 3. Локальный вариант
Для полностью бесплатного решения:
- **Ollama** + **Qwen 2.5 Coder 32B** — достаточно для базовых задач (24GB VRAM)

### Рекомендация
Если Big Pickle станет платным, попробуй сначала переключить модель в настройках opencode на бесплатные альтернативы (Claude Sonnet или GPT-4o free tier).
