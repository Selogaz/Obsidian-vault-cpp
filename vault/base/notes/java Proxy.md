**Proxy**
—
Объект-заместитель. Контролирует доступ к оригинальному объекту, позволяя выполнять дополнительные действия до или после обращения к нему.
Используется для:
- контроль доступа
- ленивая загрузка
- кэширование
- удаленный прокси
