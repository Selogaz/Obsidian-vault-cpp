**Освещение в играх и nvidia freestyle**
—
# Развитие света

- изначально освещение было вшито в текстуры
- затем появился lightmap - карта, которая характеризует требования к высветляемости пикселей и степень высветления. Стало возможно динамическое освещение. lightmap - вычисляемая вещь
- global illumination - глобальное освещение. Все объекты, на которые попадает свет, этот свет рассеивают в доступные для них стороны.

reshade, nvidia freestyle - добавление опций графики в игру. Любой движок взаимодействует с железом с помощью API (DirectX, Vulkan). Nvidia freestyle позволяет вклиниться в это взаимодействие и менять запрос движка([[java Proxy|Proxy]]?)

# Развитие теней

SSAO - screen space ambient occlution - технология не про свет, а про тени.
- врисованные в текстуры тени
- полупрозрачные текстуры проекций одних объектов на другие
	- правильная тень в зависимости от геометрии объектов, на которые тень падает
	- размытие контуров теней(PCSS)
- затенение всего окружения
	- Ray Traced Shadows
	- Ray Traced AO

SSAO - тень находится в тех местах, где наибольшее изменение геометрии(углы/стыки предметов друг с другом). Определить эти места можно по карте глубины. Карта глубины - характеризует параметры удаленности объектов в кадре от виртуальной игровой камеры.

SSAA позволяет применять анизотропную фильтрацию там, где она не была предусмотрена![[Трассировка лучей теперь доступна во всех играх? 2025-10-24.png]]

# Ограничения

Трассировка работает только с уже отрисованными(статичными) объектами. А значит не работает со статическим(вжатым в текстуры) светом и тенью.
