**Литералы**
—
Значения, хранящиеся в переменных или константах
```java
class Test {
int a = 0b1101010110;
public static void main(String[] args) { System.out.println("Hello world!");
	}
}
```
здесь `0b1101010110` - литерал
`"Hello world!"` - тоже литерал
