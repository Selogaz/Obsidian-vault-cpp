**Типы классов в java**
—
- top level (обычные классы)
	- abstract class
	- final class
- interfaces
- enum
- nested class
	- static nested
		- независим от внешнего
		- доступ только к static внешнего
	- member inner
		- связан с экземпляром внешнего
		- доступ ко всем полям внешнего
	- local inner
		- внутри метода
		- виден только в этом методе
	- anonymous inner
		- класс без имени
