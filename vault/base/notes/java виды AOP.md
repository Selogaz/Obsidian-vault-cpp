**виды AOP**
—
- статическое
	- связывание происходит во время компиляции. Пример реализации: AspectJ
- динамическое
	- связывание во время выполнения.
	- низкая производительность
	- не требуется перекомпилировать весь код приложения
	- реализация: Spring AOP
