**взаимодействие микросервисов**
—
Микросервисы общаются между собой JSON-сообщениями в стиле REST. Могут использоваться брокеры сообщений:
- IBM MQ
- Apache Kafka
- RabbitMQ
Взаимодействие может быть организовано через [[java gRPC|gRPC]]
