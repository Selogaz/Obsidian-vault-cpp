# Схема работы авторизации

## Регистрация

```mermaid
sequenceDiagram
    participant C as Клиент
    participant S as Сервер
    participant D as БД

    C->>S: POST /auth/register<br/>{email, password}
    S->>D: Сохранить user
    D-->>S: OK
    S-->>C: 201 Created<br/>{id, email, role, createdAt}
```

## Логин

```mermaid
sequenceDiagram
    participant C as Клиент
    participant S as Сервер
    participant D as БД

    C->>S: POST /auth/login<br/>{email, password}
    S->>D: Проверить credentials
    D-->>S: OK/Failed
    S->>D: Создать сессию
    D-->>S: OK
    S-->>C: 200 OK<br/>{user: {id, email, role}}<br/>+ HttpOnly cookie
    Note right of S: Сервер сам ставит<br/>Set-Cookie: session_id
```

## Проверка авторизации (/me)

```mermaid
sequenceDiagram
    participant C as Клиент
    participant S as Сервер
    participant D as БД

    C->>S: GET /me<br/>Cookie: session_id
    S->>D: Найти сессию
    D-->>S: user data
    S-->>C: 200 OK<br/>{user: {id, email, role}}
```

## Выход (Logout)

```mermaid
sequenceDiagram
    participant C as Клиент
    participant S as Сервер
    participant D as БД

    C->>S: POST /auth/logout<br/>Cookie: session_id
    S->>D: Удалить сессию
    D-->>S: OK
    S-->>C: 200 OK<br/>Clear-Cookie: session_id
```

## Обновление сессии (Refresh)

```mermaid
sequenceDiagram
    participant C as Клиент
    participant S as Сервер
    participant D as БД

    C->>S: POST /auth/refresh<br/>Cookie: session_id
    S->>D: Проверить сессию
    D-->>S: OK/Expired
    S-->>C: 200 OK<br/>{status: "ok"}
```

## Ключевые требования спецификации

1. **Сервер сам ставит auth-cookies** — клиент НЕ участвует в установке cookie
2. **Не читать токены** — клиент НЕ читает содержимое HttpOnly cookie
3. **Не хранить токены** — клиент НЕ хранит токены в localStorage/sessionStorage
4. **credentials: 'include'** — все запросы отправляют cookies

## Текущая реализация (mock)

Без реальной БД используется in-memory массив `users` для эмуляции хранения пользователей. При подключении реального бэкенда массив заменяется на запросы к БД.
