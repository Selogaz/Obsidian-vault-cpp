Запрещает создание нескольких указателей на один адрес

```c++
#include <memory>
int *p = new int(5);
unique_ptr<int> p1(p);
p1.reset();//затирает данные и становится пустым
p1.release();//данные остаются в памяти, но указатель больше не указывает на них
```
