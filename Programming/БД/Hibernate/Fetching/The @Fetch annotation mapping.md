Помимо аннотаций FetchType.LAZY или FetchType.EAGER Jakarta Persistence, вы также можете использовать аннотацию @Fetch, специфичную для Hibernate, которая принимает один из следующих FetchModes:
SELECT
  
Ассоциация будет получена с использованием вторичного выбора для каждого отдельного объекта, коллекции или загрузки соединения. Этот режим можно использовать как для FetchType.EAGER, так и для FetchType.LAZY.  
JOIN 
  
Используйте внешнее соединение для загрузки связанных объектов, коллекций или объединений при использовании прямой выборки. Этот режим можно использовать только для FetchType.EAGER.  
SUBSELECT
  
Доступно только для коллекций. При доступе к неинициализированной коллекции этот режим выборки инициирует загрузку всех элементов всех коллекций одной роли для всех владельцев, связанных с контекстом персистентности, с использованием одного вторичного выбора.
