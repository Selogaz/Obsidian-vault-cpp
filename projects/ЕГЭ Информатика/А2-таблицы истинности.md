### 2
**Информатика А2 (3 или) найти соответствие между столбцами и переменными с помощью таблицы истинности**
![[Подготовка к ЕГЭ 2026-02-24-1.png]]
Логическая функция _F_ задаётся выражением
 (x∧¬y∧z)∨(¬x∧¬y∧¬z)∨(x∧¬y∧¬z)
На рисунке приведён фрагмент таблицы истинности функции _F_, содержащий **все** наборы аргументов, при которых функция _F_ истинна.
Определите, какому столбцу таблицы истинности функции _F_ соответствует каждая из переменных _x_, _y_, _z_.
—
**Решение:**
Заметь общую закономерность: во всех трех частях есть `-y`. Внутри скобок везде И. Значит, `-y` должен быть равен -1. А значит, y=0 в каждой строке.
<!--SR:!2026-03-28,20,250-->

Теперь возьмем x и z из любой части выражения и подставим значения из таблицы истинности. В результате выражение в скобках должно быть = 1. Например, возьмем 1 строку: х=2 столбец, а z=3 столбец. В таком случае первое выражение:
$$(x*-y*z)=1*1*0=0$$
Значит, мы выбрали неверно. И x=3 столбец, а z=2 столбец. Ответ: **yzx**

**Решение**:

$$(x∧¬y)∨(y∼z)∨w$$

Верно. Значит _y - крайний правый столбец_

#### Анализ строки 1 (1???)

Проверим:
$$(0∧1)∨(0∼1)∨0 = 0∨0∨0=0$$
Верно. Значит, _z - первый столбец_

### Импликация
![[Подготовка к ЕГЭ 2026-03-15.png]]
—
**→** - импликация. Ложь, когда первое 1, а второе 0. Во всех остальных случаях истина.
Крышка = И
#### Упрощаем логическую функцию:
z -> x = -z∨x
y -> (x~w) = -y ∨(x~w)
-(y→(x~w))=-(-y∨(x~w))=y∧-(x~w)
- [x] #task/inbox #category/study - Почему поменялся знак с ∨ на ∧? ✅ 2026-03-15[^1]
Итого
y∨-(x~w)∧(z->x), где
-(x~w) истинно, когда x != w (разные)
Для истинности всей функции должны выполняться условия:
1. y=1
2. x!=w = 1
3. -z∨x = 1
#### Анализ таблицы
исходя из 3 строки:
- y=1
	- y - 3 столбец
- x!=w - значит, одно равно 0, другое 1
	- в строке есть только 1 единица и это y, значит неизвестное значение должно быть 1 (чтобы x или w было 1)
- Строка 3 полностью: 1010
#### Соответствие переменных
Строка 3:
- 3 = y
- Пусть C1 = x, тогда
- y=1
- x!=w=1
- -z∨x=-0∨1=1
- F=1
Строка 1:
- C1-?, C2=1, C3=1, C4-?
- C3 = y
- x!=w:x!=1 -> x=0 -> C1 = 0
Строка 1: **C1=0, C2=1, C3=1, C4=0**
Проверка: x=0, w=1, y=1, z=0
- y = 1 ✓
- x ≠ w: 0 ≠ 1 ✓
- ¬z ∨ x = ¬0 ∨ 0 = 1 ✓
- F = 1 ✓
**Строка 2:** C1=0, C2=?, C3=?, C4=?
- x = C1 = 0
- F = y ∧ (x≠w) ∧ (¬z∨x) = C3 ∧ (0≠C2) ∧ (¬C4∨0) = C3 ∧ C2 ∧ ¬C4
- Для F=0 достаточно, чтобы хотя бы одно было 0 ✓
Все сходится!
Ответ: **xwyz**

### Решение А2 на python
¬(y→(x∼w))∧(z→x)
—
```python
print('x y z w')
for x in range(2):
    for y in range(2):
        for z in range(2):
            for w in range(2):
                if( not(y <= (x == w)) and (z <= x) ) == True:
                    print(x,y,z,w)
```
