# Description

Долгосрочный план переезда в Японию по рабочей визе. 10 лет: от нулевого японского до релокации.

Целевой путь: **Working Holiday Visa → Engineer/Humanities Visa** или **的直接採用 (shūkinkyokujin)**.

# 🟦 План по этапам

## 🟦 Японский язык

JLPT N5 → N4 → N3 → N2 → N1

Целевой уровень для рабочей визы: **N2 минимум**, N1 желательно.

Срок: непрерывно, 2026–2034.

**Стратегия:** [[Стратегия изучения японского]]

## 🟦 Карьера и профессия

Выбор индустрии → накопление опыта → позиционирование для японского рынка.

Японские востребованные индустрии: IT, Data Science, Game Dev, FinTech, Robotics.

**Стратегия:** [[Стратегия карьеры в Японии]]

## 🟥 Поиск работы в Японии

Job-сайты, нетворкинг, прямой контакт с компаниями.

Срок: 2033–2035.

## 🟥 Визовый процесс

Engineer / Humanities Visa, документы, sponsor от работодателя.

Срок: 2034–2035.

## 🟥 Переезд и адаптация

Жильё, регистрация, языковая практика, социальная интеграция.

Срок: 2035+.
