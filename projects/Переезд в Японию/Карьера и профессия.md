# Цель

К 2033 году — иметь опыт и навыки, которые японские работодатели готовы спонсировать.

# Японские востребованные индустрии

- **IT / Software Engineering** — наибольший спрос, много вакансий для иностранцев
- **Data Science / ML** — растущий рынок
- **Game Development** — Tokyo concentration
- **FinTech** — особенно после 2020 reforms
- **Robotics / Embedded** — инженеры нужны

# Фокус

Ориентир на **IT / Software Engineering** — твой текущий профиль (Python, веб-разработка, ML). Это путь с наименьшим барьером входа.

## Что нужно для позиционирования

- [ ] 5+ лет коммерческого опыта
- [ ] Портфолио / GitHub с японским README
- [ ] Опыт с японскими tech-стеком (Ruby on Rails, Japanese cloud)
- [ ] Публичный профиль на Japanese job platforms
- [ ] Японский N2 к моменту поиска работы
