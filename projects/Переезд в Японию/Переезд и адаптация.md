# Цель

Успешная адаптация после переезда. Первый год — самый сложный.

# Жильё

- **Gaijin Housing** — https://www.gaijinpot.com/housing/ — для иностранцев
- **SUUMO** — https://suumo.jp — основной сайт
- **LEopalACE** — https://leopalace21.com — готовые квартиры с поддержкой

Первые 6 месяцев: проще снять через агентство, которое работает с иностранцами. Затем — пересмотреть.

Ключевое: многие арендодатели не сдают иностранцам без поручителя или депозита 6 мес.

# Адаптация

- [ ] Регистрация в City Hall (転入届) — 14 дней после приезда
- [ ] Банковский счёт — SMBC, Mitsubishi UFJ, Post Bank
- [ ] Налоговый номер (MyNumber) — сразу после регистрации
- [ ] Мобильная связь — MVNO (LINEMO, Rakuten) для начала
- [ ] Health Insurance (国民保険) — обязательно с первого дня

# note

Японский N1 к моменту переезда сильно упрощает адаптацию. Без него — первый год будет сложно с бюрократией.
