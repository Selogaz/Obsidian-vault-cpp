# План действий

## Немедленные меры (сейчас)

1. ⚠️ **Избегать любой физической нагрузки** — пока симптомы острые
2. ✅ Продолжать Атаракс (назначение невролога)
3. ✅ Продолжать согревающие перчатки/носки
4. [ ] Начать контрастный душ (осторожно)
5. [ ] Начать дыхательные практики

## Медицинская помощь (1-2 недели)

- [ ] Записаться к **неврологу** (повторно)
- [x] Записаться к **психотерапевту** / психологу ✅ 2026-03-16
- [ ] Контроль давления (было под 170 — наблюдать)

## Анализы (если будут деньги)

- [x] Витамин D ✅ 2026-03-30
- [x] Железо, ферритин ✅ 2026-03-30
- [x] Магний ✅ 2026-03-30
Когда появится возможность, сдайте:

1. **Ферритин** (обязательно, даже если гемоглобин в норме)
2. **Сывороточное железо + ОЖСС** (общая железосвязывающая способность)
3. **Витамин D (25-OH)**
4. **Магний** (лучше в эритроцитах, но можно и сыворотка)

Дополнительно (если будет возможность и желание врача):

- **В12, фолиевая кислота** (после антибиотиков и стресса часто истощены)
- **ТТГ, Т3св, Т4св** (щитовидная железа — исключить дисфункцию, которая может давать тревогу и тахикардию)
## Долгосрочное

- Работа со стрессом (психотерапия)
- Дыхательные практики
- Закаливание (когда стабилизируется)
- Контроль через дневник наблюдений
