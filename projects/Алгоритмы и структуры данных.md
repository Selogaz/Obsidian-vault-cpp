# Карточки
Печатать на экран "Yes", если деление числа на слагаемые будет давать четные числа и "No", если нечетные.
—
```java
import java.util.Scanner;  
  
public class Main {  
    public static void main(String[] args) {  
        Scanner input = new Scanner(System.in);  
        int a = input.nextInt();  
        System.out.println(yesIfEven(a));  
        }  
    public static String yesIfEven(int a) {  
        if (a % 2 == 0 && a >=4) {  
            return "Yes";  
        } else {  
            return "No";  
        }  
    }  
}
```

Слова длинной > 10 превращать в "1символ + количество букв между 1 и последним символом + последний символ". Первой строкой будет число, указывающее количество слов, которые последуют каждый с новой строки
—
```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        int a = input.nextInt();
        for (int i =0; i<=a;i++) {
            String s = input.nextLine();
            if (s.length() > 10) {
                String newStr = s.charAt(0) + String.valueOf(s.length() - 2) + s.charAt(s.length()-1);
                System.out.println(newStr);
                continue;
            }
            System.out.println(s);
        }
    }

}
```

DBMB and the Array
![[Алгоритмы и структуры данных 2026-03-09.png]]
—
первое число - n, второе - s, третье - x
n - количество чисел во второй строке
s - число, которой должна быть равна сумма чисел во второй строке. Если этого(числа s) можно достичь, прибавляя x к одному из чисел во второй строке, то выводить YES. Иначе NO
Также s должно быть меньше суммы чисел.
```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        int cycleCounter = input.nextInt();
        input.nextLine();//для перехода на новую строку, ведь nextInt() не считывает символ конца строки
        while (cycleCounter > 0) {
            int sum = 0;
            int n = input.nextInt();
            int s = input.nextInt();
            int x = input.nextInt();
            input.nextLine();

            int[] arr = new int[n];
            String[] secondLine = input.nextLine().split(" ");
            for (int i = 0; i < n; i++) {
                arr[i] = Integer.parseInt(secondLine[i]);
                sum += arr[i];
            }

            if (sum == s) {
                System.out.println("YES");
            } else if (((s - sum) % x == 0) && (sum < s)) {
                System.out.println("YES");
            } else {
                System.out.println("NO");
            }
            cycleCounter--;
        }
    }
}
```
