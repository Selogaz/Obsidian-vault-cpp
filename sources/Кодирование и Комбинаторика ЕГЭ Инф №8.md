# Три формулы комбинаторики

## Формула перестановки

**Все** элементы используются, порядок **важен**
$P_n = n!$, где
n - количество элементов, которые можно использовать
![[Кодирование и Комбинаторика ЕГЭ Инф №8 2026-03-30.png]]

## Формула размещения

Используется **часть** элементов, порядок **важен**
$A_n^k = \frac{n!}{(n-k)!}$, где
$k$ - то количество элементов, которое **можно** взять
$n$ - общее количество элементов
![[Кодирование и Комбинаторика ЕГЭ Инф №8 2026-03-30-1.png]]

## Формула сочетания

Используется **часть** элементов, порядок **не важен**
$C_n^k = \frac{n!}{k!(n-k)!}$, где
$k$ - то количество элементов, которое **можно** взять
$n$ - общее количество элементов
![[Кодирование и Комбинаторика ЕГЭ Инф №8 2026-03-30-2.png]]

# Библиотека itertools

```python
from itertools import product, permutations
```

## Permutations

Используется, когда нужно, чтобы символы **не повторялись**

```python
Permutations("Доступные символы", r = длина слова)
```

```python
words = permutations('AБВ', r = 3)
for w in words:
	print(w)
```
![[Кодирование и Комбинаторика ЕГЭ Инф №8 2026-03-30-3.png]]

## Product

Используется когда нужно, чтобы символы **повторялись**

```python
Product("Доступные символы", repeat = длина слова)
```

```python
words = product('AБВ', repeat = 2)
for w in words:
	print(w)
```

# Примеры задач

Определите количество семеричных шестизначных чисел, которые начинаются с четных цифр и содержат ровно две цифры 5
—
- Какую функцию будем использовать?
	- Не указаны ограничения, заставляющие использовать символы только один раз, так что будем использовать `Product`
- Есть ли дополнительные ограничения в задаче, не описанные в условии?
	- Есть. Число не может начинаться с 0
```python
from itertools import *

cnt = 0
words = product('0123456', repeat = 6)
for w in words:
	if (w[0] in '246') and (w.count('5') == 2):
		cnt+=1
	print(cnt)
```

Маша составляет слова из семи букв, в которых есть только буквы из слова КАБИНЕТ. Каждая буква используется в каждом слове 1 раз, слово не может оканчиваться на гласную. Словом считается любая допустимая последовательность букв, не обязательно осмысленная. Сколько существует слов, которые может написать Маша?
—
```python
from itertools import *

cnt = 0
words = permutations('КАБИНЕТ', r = 7)
for w in words:
	if w[-1] not in 'АИЕ':
		cnt+=1
	print(cnt, w)
```

Игорь составляет таблицу кодовых слов для передачи сообщений, каждому сообщению соответствует своё кодовое слово, В качестве кодовых слов Игорь использует пятибуквенные слова, в которых есть только буквы Э,Т,А,Н, причем, в каждом слове есть только одна гласная и она встречается ровно 1 раз. Каждая из допустимых согласных букв может встречаться в кодовом слове любое количество раз или не встречаться совсем. Сколько различных кодов может использовать Игорь?
—
- Какую функцию использовать?
	- `Product`. Есть "любое количество раз" в предпоследнем предложении.
```python
from itertools import *

cnt = 0
words = product('ЭТАН', repeat = 5)
for w in words:
	if (w.count('А') == 1 and w.count('Э') == 0) or (w.count('Э') == 1 and w.count('А') == 0):
		cnt+=1
	print(cnt)
```

Все четырехбуквенные слова, в составе которых могут быть буквы А,В,Л,О,С записаны в алфавитном порядке и пронумерованы, начиная с 1. Ниже приведено начало списка:
- АААА
- АААВ
- АААЛ
- АААО
- АААС
- ААВА
Под каким номером идет первое слово, начинающееся с буквы Л?
—
- Какую функцию использовать?
	- Очевидно, `Product`. Явно ограничения не указаны, плюс в примере явно есть повторы символов.
```python
from itertools import *

cnt = 0
words = product('АВЛОС', repeat = 4)
for w in words:
	cnt+=1
	if w[0] == 'Л':
		print(cnt,w)
		break
```
